# Abstract Data Types

[![Build Status](https://travis-ci.org/ljrocha/AbstractDataTypes.svg?branch=main)](https://travis-ci.org/ljrocha/AbstractDataTypes)

A Swift package for common abstract data types (ADTs).

## Abstract Data Types
- Stack
- Queue
- Linked List

## Requirements
- Swift 5
